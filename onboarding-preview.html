~ Design Notes ~

🎨 Color Progression

Each screen uses a different accent color (Coral → Sage → Lavender → Sunny) to create visual variety while maintaining warmth.

✨ Floating Elements

Subtle floating decorations add playfulness and movement without being distracting. They reinforce each screen's theme.

📝 Typography

Caveat font for titles creates a personal, handwritten feel. Nunito for body text ensures readability and friendliness.

🔘 Navigation

Skip option respects user's time. Animated pagination dots show progress. Clear CTA buttons guide the journey.
